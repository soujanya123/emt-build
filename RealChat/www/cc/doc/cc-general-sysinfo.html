System Info

 

The System Info page allows you to enter your registration key after you purchase RealChat. You can also use this page to check to see if any updates are available for RealChat. Finally, the system info page displays information about the server which RealChat is running on.

Registering RealChat

To Register the program, select the Register/Upgrade button and enter the registration code which you received. After entering the registration information, click the Submit button.

Checking for Updates

RealChat periodically checks for and automatically downloads updates for you. To manually check for updates, simply click the Check for Updates button.
