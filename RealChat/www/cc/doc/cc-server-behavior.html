Behavior Control

 

The behavior control page gives you control over several automatic features which help you to control problem users. Each feature is described below.

Max Message Length

The maximum message length control allows you to control how many characters can be typed and then sent to the chat server at one time.

Default Ban Timeout

This setting defines the default ban timeout in minutes. When a user has been kicked out of a chat room, they cannot rejoin the conversation until the ban times out. The ban timeout value applies to the following situations:

Users automatically kicked for flooding.
Users automatically kicked for profanity.
Users kicked by operator/administrator.

Kick URL

This setting is an optional URL which users are redirected to after they have been kicked out of a chat sever either by the automatic functionality or manually by an operator or administrator. The full url should be provided. I.e. http://www.yourdomain.com/sitepolicies.htm

We suggest that this page that explains your chat community policies as well as why users may be kicked or banned.

Warn and Kick Users who are flooding

When enabled, this setting will cause the chat server to automatically warn and then kick any user that rapidly enters chat messages without pause.

Warn and Kick Inactive Users

When enabled, this setting will cause the chat server to automatically warn and then kick any user that has not typed a message for the specified time period.

Profane Word Filter

When you enable the profane word filter, the Control Center will display two additional controls which allow you to control which words are filtered and what consequences are applied to users who are repeat offenders.

Warn then Kick Users who use profane words

When this setting is enabled, the chat server will automatically warn and then kick any user that uses profanity more than the specified number of times. If this setting is not used, the profanity will simply be replaced with dashes, but users will not be kicked off.
