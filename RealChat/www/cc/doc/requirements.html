System Requirements

Server-side

To run the RealChat Software on your server machine you need:

  • A dedicated server or unrestricted shell account on a shared server.
  • Java Runtime Environment installed on the server.
  • UNIX, Linux or Windows operating system.

Client-side

End-users only need a web browser with Flash installed to participate in an online chat room.
