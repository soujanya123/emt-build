Usage Monitor

 

The usage monitor section allows you to monitor the chat server and gives you information about the number of users that are currently using the chat server, how many messages are being sent, where users are located, and any bans that are in effect. The Usage Monitor Section is divided into the following pages:


  • Dashboard - allows you to monitor the number of users on your chat server as well as the number of messages being handled by the chat server.
  • Room Browser - allows you to view the available rooms as well as what users are in each room.
  • Ban List - The ban list shows you a list of what bans are currently active with information about when the ban is lifted and why the ban was placed. You can also lift individual bans or all bans.
