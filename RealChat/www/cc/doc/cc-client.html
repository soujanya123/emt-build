Client Settings

 

The Client Section allows you to create links to allow your users to access the chat server and define how the chat client looks and what functionality is available within the chat client. The Client Section is divided into the following pages:


  • Code Generator - allows you to create urls that can be used to access the chat server.
  • Client Profiles - allows you to define what functionality is available within the client interface.
  • Color Themes - allows you to customize the look and feel of the client interface to match the look of the remainder of your website.
