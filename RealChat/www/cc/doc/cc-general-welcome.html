Welcome

 

The Welcome page displays general information about RealChat and includes information about how to display this help manual and how to receive help if you have any questions about using RealChat.

If RealChat has just been installed, the bottom of the welcome page will also include information about the remaining steps that should be completed to setup and secure your chat server. Clicking on a step will take you the correct panel to complete that step. After you complete each step, the step will be crossed out. For more information on setting up your website, see the Basic Server Configuration topic.

After you have successfully configured the server, you may want to select the When starting, go directly to the Room Browser checkbox. This will cause RealChat to bypass the Welcome page when you open the Control Center. Instead, you will be taken directly to the Room Browser.
