Manage Your Chat Rooms

 

RealChat installs two sample rooms by default. These rooms are called Lobby and Staff Only. In most cases, you will want to either modify these rooms, or add additional rooms.

Adding a New Room

To add a room, use the Room Settings page and select the Add link.

Editing Room Properties

To edit a room, you use the Room Settings page. First select the room which you want to edit in the Room List.

After you have selected the room you wish to edit, set a room name which describes the room. You should also define a topic which is meaningful to your users. For example, if you want to create a room where your users can discuss poker strategy, you could call the room "Poker Strategy" and the topic to: "Discuss how to bluff other players and win at poker".

After you define the room name and topic, you need to determine who should be able to access the room. For example, if you have a chat room dedicated to internal projects for your company, you would not want the chat room to be exposed to the general public. There are two methods protecting the chat room. The first is defining a password for the room. This password should only be made available to users you trust. The second method of protection is making the chat room invisible. If the room is invisible, it must be accessed by a direct url. You can create the url using the Code Generator. You can also use direct integration to allow your users to access the hidden rooms.

Deleting a Room

Deleting a room is also done on the Room Settings page. First select the room which you want to delete in the Room List and then click on the Remove Link.

Controlling the Maximum Number of Users in a Room

To control the maximum number of users that can be in a room at any time, use the Users per room setting on the Server Settings page. Please be aware that this is the maximum number of users that can be in any room at one time. If the maximum number of users is reached a 'spare' room is created automatically.

Letting Users Change Rooms and Create Rooms

You can determine whether or not users can change rooms using profiles. You configure profiles on the Client Profiles page. You disable the ability to switch rooms using the Switch Rooms setting. If you want your users to be able to change between rooms, but you do not want them to be able to create private rooms, you can disable the Create Rooms setting.
